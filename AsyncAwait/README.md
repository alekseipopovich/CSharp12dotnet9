# Асинхронность

- Асинхронная загрузка данных (LoadGradesAsync):

  - Используется FileStream для чтения данных из файла.

  - Метод JsonSerializer.DeserializeAsync асинхронно десериализует данные из JSON в список объектов Grade.

  - Если файл не существует, возвращается пустой список.

- Асинхронное сохранение данных (SaveGradesAsync):

  - Используется FileStream для записи данных в файл.

  - Метод JsonSerializer.SerializeAsync асинхронно сериализует список объектов Grade в JSON.

- Использование await:

  - В методе Main используется await для асинхронного вызова методов LoadGradesAsync и SaveGradesAsync. Это позволяет не блокировать основной поток при выполнении операций ввода-вывода.

- Обработка ошибок:

  - В обоих методах добавлена обработка исключений для предотвращения сбоев приложения.

Преимущества асинхронности:

- Отзывчивость UI:

  Если бы загрузка и сохранение данных выполнялись синхронно, интерфейс приложения мог бы "зависнуть" на время выполнения этих операций. Асинхронность позволяет избежать этого.

- Эффективность:

  Асинхронные операции позволяют эффективно использовать ресурсы системы, особенно при работе с файлами, сетью или базами данных.

- Масштабируемость:

  Асинхронный код легче масштабировать для выполнения множества операций одновременно.
